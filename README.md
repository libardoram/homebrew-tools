# Homebrew Tap for libardoram's Tools

This is a Homebrew tap for installing libardoram's command-line tools.

## Installation

```bash
# Tap this repository
brew tap libardoram/tools

# Install macnetmonitor
brew install macnetmonitor
```

## Available Formulae

- **macnetmonitor** - macOS Network Monitor TUI (command: netw)

## Update

```bash
brew update
brew upgrade macnetmonitor
```
